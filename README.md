# Резидентура ИИ-лаборатории

Лендинг закрытого платного сообщества для продвинутых пользователей ИИ — раздел в стиле сайта [glebkudr.com](https://glebkudr.com).

**Живая версия:** https://shao3dnexx.github.io/ai-lab-residency/

---

## Что это

Раздел-лендинг под бренд **«ИИ-лаборатория Глеба Кудрявцева»**, реализующий концепцию из ТЗ: закрытое сообщество практиков ИИ с входом по собеседованию, принципом вовлечённого участия и позиционированием «стать студентом лаборатории».

Сделано в визуальном стиле основного сайта (тёмная «лабораторно-терминальная» эстетика, оранжевый акцент `#fe6e00`, шрифты Unbounded + Commissioner, чередование тёмных и кремовых секций).

## Ключевые продуктовые решения (на основе рыночного ресёрча)

ТЗ дано кратко, поэтому продукт «доресёрчен» по реальному рынку платных AI/профессиональных сообществ (South Park Commons, On Deck, Reforge, Hampton, Recurse Center; российские бизнес-клубы Эквиум, Атланты, Club 500; Нейроклуб, Epic Growth и др.).

- **Позиционирование:** не курс и не чат, а **резидентура при лаборатории**. Идентичность «резидент лаборатории» (по образцу On Deck «fellow» / Recurse Center «Never Graduate»).
- **Принцип вовлечённого участия → двухтарифная модель.** Активность снижает цену, пассивность повышает:
  - **Резидент — 10 000 ₽/мес** (+ вклад: дипломный проект, спикерство ≥1×/6 мес, организация событий, волонтёрство). Платит меньше, потому что создаёт ценность трудом (логика sweat-equity).
  - **Наблюдатель — 25 000 ₽/мес** (без обязательств). Платит больше, потому что выкупает обязательство вклада деньгами. Чистая кратность ×2,5 делает фрейм «платишь за двоих» интуитивным.
- **Ценовое «белое пятно»:** 10 000 ₽/мес попадает между дешёвым AI-контентом (Нейроклуб ~3 900 ₽) и дорогими клубами с отбором (450 000–820 000 ₽/год). Отбор + диплом за десятую часть цены бизнес-клуба.
- **Вход по собеседованию** как знак уровня: «ценят только тот отбор, который реально отказывает».
- **Дипломный проект** — центр продукта: осязаемый результат вместо «архива записей». IP остаётся за участником.
- **Еженедельный живой формат:** разбор темы / мастер-класс / приглашённая звезда. Темы переднего края 2025–2026: агенты и надёжность, evals, MCP, agentic RAG, GRPO/agentic RL, context engineering, локальные модели.

Полные материалы ресёрча — в `_research/` (бриф, источники; не публикуется).

## Стек

Статическая страница, без зависимостей: HTML + CSS + минимальный JS.

| Файл | Назначение |
|------|-----------|
| `content.json` | Весь текст лендинга (результат ресёрча + копирайтинга) |
| `build.js` | Генератор: `content.json` → `index.html` (Node, без зависимостей) |
| `index.html` | Собранная страница (коммитится) |
| `styles.css` | Дизайн-система в стиле glebkudr.com |
| `app.js` | FAQ-аккордеон, мобильное меню, анимация появления |

## Сборка и локальный запуск

```bash
node build.js                 # пересобрать index.html из content.json
python3 -m http.server 8765   # открыть http://localhost:8765
```
