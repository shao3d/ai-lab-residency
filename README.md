# Резидентура ИИ-лаборатории

Лендинг закрытого платного сообщества для продвинутых пользователей ИИ, в визуальном стиле [glebkudr.com](https://glebkudr.com). Тестовое задание.

**Живая версия:** https://shao3d.github.io/ai-lab-residency/

---

## Как читать этот репозиторий

ТЗ дано намеренно кратко. Я отнёсся к нему не как к «нарисуй лендинг», а как к продуктовой задаче и прошёл её насквозь:

1. **Ресёрч** — «доресёрчил» продукт по реальному рынку платных AI/проф-сообществ → [`ИССЛЕДОВАНИЕ.md`](ИССЛЕДОВАНИЕ.md)
2. **Сборка (v1)** — построил полную, развёрнутую интерпретацию брифа: живой лендинг (этот репозиторий, задеплоен)
3. **Состязательный стресс-тест** — прогнал концепцию через разбор по 17 вопросам, независимую внешнюю рецензию и факт-чек по живым источникам
4. **Рекомендация (v2)** — что я бы реально запустил вместо максималистской версии и почему → [`РЕКОМЕНДАЦИЯ.md`](РЕКОМЕНДАЦИЯ.md)

> **v1 — широкая разведка решения; v2 — узкая рекомендация, которую она заработала.** Лендинг доказывает исполнение, записка — суждение.

## Что построено (v1)

Раздел-лендинг под бренд «ИИ-лаборатория Глеба Кудрявцева» в тёмной «лабораторно-терминальной» эстетике (оранжевый `#fe6e00`, шрифты Unbounded + Commissioner, чередование тёмных и кремовых секций). Логика решений — полное обоснование в [`ИССЛЕДОВАНИЕ.md`](ИССЛЕДОВАНИЕ.md):

- **Позиционирование:** резидентура при лаборатории, а не курс/чат.
- **Принцип вовлечённого участия → двухтарифная модель:** вклад снижает цену (Резидент 10 000 ₽ с обязательствами / Наблюдатель 25 000 ₽ без).
- **Вход по собеседованию** с named-куратором как знак уровня.
- **Дипломный проект** в центре продукта; еженедельный живой формат.

## Честно: что бы я изменил перед запуском

Максималистская v1 — сильная демонстрация диапазона, но не то, что я бы выкатил в продакшн. Коротко: инвертированный двухтариф как *центр* — умная механика, но слишком сложная публичная архитектура; высокоинтенсивный продукт продаётся слишком дёшево; часть цифр ресёрча не выдержала факт-чека (исправлено и помечено). Полный разбор и пилот-рекомендация — в [`РЕКОМЕНДАЦИЯ.md`](РЕКОМЕНДАЦИЯ.md).

## Стек

Статическая страница без зависимостей: HTML + CSS + минимальный JS.

| Файл | Назначение |
|------|-----------|
| `content.json` | Весь текст лендинга |
| `build.js` | Генератор `content.json` → `index.html` (Node, без зависимостей) |
| `index.html` | Собранная страница (коммитится) |
| `styles.css` | Дизайн-система в стиле glebkudr.com |
| `app.js` | FAQ-аккордеон, мобильное меню, анимации |

## Сборка и деплой

```bash
node build.js                 # пересобрать index.html из content.json
python3 -m http.server 8765   # открыть http://localhost:8765
```

Деплой — GitHub Pages из ветки `main` (корень репозитория); `.nojekyll` отключает Jekyll.
